# AI Email Automation System

An AI-powered email classification and automation system built using FastAPI, SQLite, HTML, CSS, and JavaScript.

This project analyzes customer emails, categorizes them, assigns priority levels, generates automated replies, and stores email history in a database.

---

## Features

- AI-style email analysis
- Email category detection
- Priority classification
- Automated response generation
- SQLite database integration
- Email history tracking
- FastAPI backend API
- Interactive frontend dashboard
- Swagger API documentation

---

## Tech Stack

- Python
- FastAPI
- SQLite
- HTML
- CSS
- JavaScript
- Uvicorn
